本游戏是以PLASTIC组织的成员为角色制作的一款~~开源~~RPG游戏。

# 内容
主角周明凯在前妻死亡后努力奋斗成为冰封王国的大勇士。<br/>
为救出被抓走的奶茶公主，前往冒险，一路奋斗，结识伙伴，达成梦想。<br/>

### 当前出场人物
